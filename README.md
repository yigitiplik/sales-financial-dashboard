# Sales & Financial Performance Dashboard

This project demonstrates an end-to-end Power BI dashboard built for analysing sales, revenue, costs, and profitability metrics.
It replicates a real-world business scenario and shows how data can be transformed into actionable insights.

# Key Features

* Sales revenue tracking
* Profit & margin analysis
* YoY / MoM trends
* Top customers & product performance
* Region-based performance
* KPI summary cards for executives
* Automated refresh using Power Query

# Technologies Used

* Power BI
* SQL
* Power Query


# Project Structure

/data          → raw & cleaned datasets  
/powerbi       → .pbix dashboard file  
/sql           → SQL queries used for transformations  
/screenshots   → exported dashboard images

# Business Value

This dashboard helps organisations:

* Understand customer profitability
